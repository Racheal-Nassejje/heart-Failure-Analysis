#  MedAnalytics - Health Data Analysis Dashboard

##  Overview
MedAnalytics is an interactive web dashboard that 
allows medical professionals to upload any health 
CSV dataset and instantly get automated data cleaning,
statistical analysis and visual insights.

##  Main Objective
To empower medical professionals to make faster and 
smarter clinical decisions by turning raw health data 
into instant visual insights without requiring any 
technical skills.

##  Problem Statement
Medical professionals struggle to quickly analyse 
large volumes of patient data. MedAnalytics solves 
this by automatically cleaning, analysing and 
visualising any uploaded health dataset instantly.

##  How It Works
1. User uploads any health CSV dataset
2. Dashboard automatically cleans the data
3. Removes duplicates and handles missing values
4. Calculates key statistics automatically
5. Generates interactive charts instantly
6. Displays full searchable patient table

##  Target Users
### Primary Users

- Medical Researchers
